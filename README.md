# Weather Application

A modern React-based Weather Application built with Vite.

## Features
- Real-time weather data
- Forecast charts
- Interactive dashboard

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example` and add your API keys.

### Running the App
```bash
npm run dev
```
